## 📖 O projekcie
Ten projekt został stworzony w celu przetestowania automatyzacji przepływu pracy (workflow) między popularnymi narzędziami deweloperskimi a komunikatorami biznesowymi. 

Głównym celem jest:
1. Automatyczne powiadamianie o zmianach w kodzie (**GitHub** -> **Teams**).
2. Zarządzanie zadaniami bezpośrednio z poziomu czatu (**Jira** -> **Teams**).
3. Publikacja dokumentacji technicznej przez **GitHub Pages**.
